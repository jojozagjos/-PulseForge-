How to use the Editor

Use the Tools tab to select, create, delete, stretch, copy and paste notes. The Audio tab loads song audio, and the Chart tab opens/saves chart JSON. The VFX tab creates visual effects and camera animations.

Chart Editor

  • Alt+Click sets the playhead. Middle drag pans. Shift adds to selection.
  • Use the Zoom slider or Shift+Wheel to zoom in/out; Follow keeps the playhead centered.
  • Stretch turns a note into a hold; drag its tail to adjust duration.
  • Scrub the timeline with the slider below the canvas to preview.

VFX Editor

  • Categories: Background, Camera, Notes, and Lanes each have their own property groups.
  • Keyframes: Pick a category first (Background/Camera/Notes/Lanes). Change a control (color/slider/toggle) to set the active property automatically, choose an easing, move the scrubber/playhead to the time, then click "Add Keyframe". Subsequent changes auto-keyframe that property at the playhead.
  • Timeline: Click to set the playhead and add keyframes at that time. (Drag-to-move coming soon.) Click a keyframe dot to select; use Delete Selected to remove.
  • Easing: Instant = snap change, Linear = constant speed, Ease In/Out = smooth acceleration/deceleration.
  • Background: Set colors, gradients, and beat-synced flash effects.
  • Camera: Control position (X,Y), Z-depth zoom, Rotate X/Y (2.5D skew), Rotate Z (spin), and camera shake.
  • Notes: Customize individual lane colors, glow effects, size, and trails.
  • Export: Save VFX settings as a .json file that can be loaded into the game runtime.

VFX Workflow

  1. Load your chart and audio in the Chart and Audio tabs.
  2. Switch to the VFX tab and select a category (Background, Camera, Notes, or Lanes).
  3. Adjust the property values using sliders, color pickers, and toggles.
  4. The last control you changed becomes the active property. Choose an easing.
  5. Set the playhead to your desired time and click "Add Keyframe". Changes to that control will auto-keyframe at the playhead time.
  6. Move to another time, change the property value, and add another keyframe.
  7. Choose the easing type for smooth transitions between keyframes.
  8. Use "Preview VFX" to see your effects or "Test in Game" for full gameplay testing.
  9. Export your VFX file when finished to use in the game runtime.
